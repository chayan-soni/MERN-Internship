# Day 1 MERN Stack Internship Assignment


This repository contains my Day 1 internship assignment. The goal of this assignment was to practice some basic concepts of modern JavaScript (ES6+) and front-end development using HTML and CSS.

## Tasks Completed

### Task 2.1 - JavaScript Data Processing

In this task, I worked with a mock JSON dataset containing user information. The data was parsed, filtered, and cleaned based on given conditions.

Concepts used:

* JSON.parse()
* Arrow functions
* filter()
* map()
* Template literals
* Destructuring
* Spread operator

### Task 2.2 - Responsive User Profile Card

In this task, I created a responsive user profile card using HTML and CSS. The layout adjusts properly on both desktop and mobile screens.

Concepts used:

* Semantic HTML
* CSS Variables
* Flexbox
* Media Queries
* Responsive Design
